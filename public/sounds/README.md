# Notification sounds

Add your interpreter notification sound here:

- **Filename:** `notification.wav` (or `notification.mp3`)
- **Full path:** `public/sounds/notification.wav`

The sound plays when a new interpretation offer arrives. Use a short, clear tone (e.g. 0.5–2 seconds). MP3 is widely supported; WAV also works.

If the file is missing, the app falls back to a built-in beep.
